## 事前準備
1. LibreOfficeのインストール

2. officeHomeにパスを通す
Windowsでの設定の注意点：soffice.binは「program」配下にあるがつける必要はない
`C:\\Program Files\\LibreOffice`
※macの場合は以下
`/Applications/LibreOffice.app/Contents`
